#CodSoft Java Projects 🖥️

#A collection of Java console-based projects built during CodSoft internship.

##📁 Projects

###1. Number Game
- Generates a random number and asks user to guess it
- Gives hints (too high / too low)

### 2. Student Grade Calculator
- Takes marks as input
- Calculates total, average and grade

### 3. ATM Interface
- Simulates basic ATM operations
- Deposit, Withdraw, Check Balance

## 🔧 Tech Stack
- Java

## ⚙️ How to Run
1. Clone the repo
2. Open in any Java IDE (IntelliJ / Eclipse / VS Code)
3. Run the individual `.java` files
